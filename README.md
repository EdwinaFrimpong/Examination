# Dream Villa VR (A-Frame)

This is a lightweight A-Frame VR scene showcasing a dream villa built from provided GLB assets and simple primitive furniture.

Quick start

1. Open `index.html` in a browser that supports WebGL (Chrome/Edge/Firefox).
2. Use mouse or headset to look around; `WASD` moves the camera.

Notes
- Uses the GLB files in `assets/` (villa, pool, palms, car, gate).
- `js/main.js` contains components: `pool-water`, `pet-bob`, `floaty`, and `optimize-gltf`.
- The pool water is animated with a canvas texture (lightweight custom animation).
